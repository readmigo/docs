## 1. Feedbooks

### 1.1 状态：已关闭

| 项目 | 描述 |
|------|------|
| **原网站** | feedbooks.com（已关闭） |
| **关闭时间** | 2024年初 |
| **现状** | 域名重定向至 Cantook.com |
| **原定位** | 电子书平台，含公共领域书籍分区 |

### 1.2 历史价值

Feedbooks 曾是 OPDS 协议的主要贡献者和示范实现：
- 参与制定 OPDS 1.0 规范
- 提供标准的 OPDS 目录实现
- 公共领域书籍格式化质量较好

### 1.3 数据备份

社区成员创建了公共领域书籍备份：

| 资源 | 链接 |
|------|------|
| **GitHub 备份** | [ManuelFte/Feedbooks-public-domain-books](https://github.com/ManuelFte/Feedbooks-public-domain-books) |
| **内容** | EPUB 格式的公共领域书籍 |
| **状态** | 静态存档，不再更新 |

## 5. Faded Page (加拿大)

### 5.1 概述

| 项目 | 描述 |
|------|------|
| **网站** | https://www.fadedpage.com |
| **定位** | 加拿大公共领域电子书 |
| **特点** | 加拿大版权法（死后50年），书籍比美国更多 |

### 5.2 版权优势

加拿大版权法规定作者去世50年后作品进入公共领域，因此：
- 1973年前去世的作者作品可用
- 比美国（1928年前出版）范围更广

## 8. 推荐优先级

### 8.1 核心数据源（必须使用）

| 优先级 | 资源 | 用途 |
|--------|------|------|
| **P0** | Standard Ebooks | 高质量 EPUB，核心内容 |
| **P0** | Project Gutenberg | 大量经典，扩展内容 |
| **P0** | LibriVox | 有声书配套 |

| 优先级 | 资源 | 用途 |
|--------|------|------|
| **P1** | Open Library | 书籍描述、作者信息、封面 |
| **P1** | Wikipedia | 作者/书籍背景信息 |

## 5. 交互手势设计

### 5.1 手势映射表

| 手势 | 阅读模式 | 学习模式 |
|------|----------|----------|
| 单指左右滑动 | 翻页 | 切换卡片 |
| 单指上下滑动 | 滚动 | 滚动 |
| 双指捏合 | 缩放字体 | 缩放内容 |
| 单指长按 | 选中文本 + AI 解释 | 编辑/删除 |
| 双指长按 | 进入批注模式 | - |
| 从左边缘右滑 | 显示目录 | 返回上级 |
| 从右边缘左滑 | 显示 AI 面板 | 显示详情 |
| 双击 | 退出专注模式 | 展开/收起 |
| 三指向上滑 | 显示多任务 | 显示多任务 |

### 5.2 键盘快捷键（外接键盘）

| 快捷键 | 功能 |
|--------|------|
| `Space` | 向下翻页 |
| `Shift + Space` | 向上翻页 |
| `←` / `→` | 上一章 / 下一章 |
| `Cmd + F` | 搜索 |
| `Cmd + B` | 添加书签 |
| `Cmd + N` | 新建笔记 |
| `Cmd + L` | 查词 |
| `Cmd + T` | 切换主题 |
| `Cmd + .` | 显示/隐藏 AI 面板 |
| `Esc` | 退出当前模式 |

### 6.2 电池续航优化

| 场景 | 优化措施 |
|------|----------|
| 长时间阅读 | 降低后台活动，暂停同步 |
| 夜间阅读 | OLED 真黑省电，降低屏幕亮度 |
| AI 功能 | 本地缓存常用词汇，减少网络请求 |
| 离线模式 | 预下载书籍内容，减少 CPU 唤醒 |

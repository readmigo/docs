# Readmigo 用户画像文档

> User Persona Document - 精准定义目标用户群体

---

## 1. 产品定位

**Readmigo** 是一款 **AI 原生英语阅读学习平台**，定位于电子书阅读与语言学习的交叉领域。

**核心价值主张：** *"Read any book. AI has your back."*

| 维度 | 描述 |
|------|------|
| **产品类型** | AI 辅助英语原版书阅读 + 词汇学习工具 |
| **核心差异** | 将被动阅读转化为主动学习，AI 实时辅助理解 |
| **市场空白** | Kindle 不教英语，Duolingo 不提供真实阅读体验 |

## 2. 目标市场规模

| 市场层级 | 规模 | 说明 |
|---------|------|------|
| **TAM（总可触达市场）** | 15 亿+ | 全球非英语母语的英语学习者 |
| **SAM（可服务市场）** | 3 亿+ | 有阅读英文书籍意愿的学习者 |
| **SOM（可获得市场）** | 1000 万+ | 愿意为优质阅读学习工具付费的用户 |

**首要目标市场：** 中国、日本、韩国、东南亚、拉丁美洲等非英语国家

## 3. 四大核心用户群体

### 3.1 英语学习者（Primary Segment）

| 维度 | 描述 |
|------|------|
| **潜在规模** | 10 亿+ |
| **人群特征** | 非英语母语者，希望通过阅读提升英语综合能力 |
| **年龄段** | 18-35 岁为主 |
| **典型场景** | 想读原版书但词汇量不够，读几页就因生词太多而放弃 |
| **核心痛点** | 词汇障碍、阅读速度慢、复杂句式理解困难、学了就忘 |
| **期望价值** | 边读边学、无痛提升、有成就感、词汇量可量化增长 |
| **付费意愿** | 中等，愿意为有效工具付费（$4.99/月可接受） |

---

### 3.2 文学爱好者

| 维度 | 描述 |
|------|------|
| **潜在规模** | 5000 万+ |
| **人群特征** | 对欧美文学有浓厚兴趣，追求原著阅读体验 |
| **年龄段** | 22-45 岁 |
| **典型场景** | 想读《傲慢与偏见》《了不起的盖茨比》原版，但被语言难度劝退 |
| **核心痛点** | 复杂语言结构、文学修辞难以理解、翻译版丢失原著韵味 |
| **期望价值** | 体验原汁原味的文学之美，理解作者真正的表达意图 |
| **付费意愿** | 较高，愿意为高质量内容付费 |

---

### 3.3 学生与备考者

| 维度 | 描述 |
|------|------|
| **潜在规模** | 3000 万+ |
| **人群特征** | 托福/雅思/GRE/SAT 考生、留学生、英语专业学生 |
| **年龄段** | 16-28 岁 |
| **典型场景** | 需要大量阅读英文材料提升阅读理解能力和词汇量 |
| **核心痛点** | 时间紧迫、阅读效率低、词汇记不住、缺乏系统性 |
| **期望价值** | 高效阅读、系统积累词汇、提升考试成绩、扩展学术视野 |
| **付费意愿** | 中高，教育投资意愿强 |

---

### 3.4 古典文学爱好者

| 维度 | 描述 |
|------|------|
| **潜在规模** | 2000 万+ |
| **人群特征** | 对莎士比亚、荷马史诗、古希腊罗马文学等古典作品感兴趣 |
| **年龄段** | 25-50 岁 |
| **典型场景** | 想研读莎士比亚原著，但中古英语/古英语太难 |
| **核心痛点** | 过时语言表达、历史文化背景缺失、缺乏注释和解读 |
| **期望价值** | AI 解释上下文、简化古典表达、提供历史背景知识 |
| **付费意愿** | 高，对文化内容有强付费意愿 |

## 10. UI/UX 设计原则（基于用户画像）

### 10.1 语言显示策略

由于核心用户是**非英语母语者**，建议：

| 功能 | 显示策略 |
|------|----------|
| 作者传记 | 默认双语并排显示 |
| 金句/名言 | 英文原文 + 中文翻译（可折叠） |
| 书名 | 英文 + 中文对照 |
| 界面文案 | 根据系统语言自动切换 |
| 时间线 | 英文标题 + 中文副标题 |

### 10.2 阅读辅助等级

| 用户水平 | 推荐模式 | AI 辅助程度 |
|----------|----------|-------------|
| 入门级 | Easy Mode | 高（自动简化、全文翻译） |
| 中级 | Standard | 中（按需解释） |
| 高级 | Challenge | 低（仅词典查询） |

### 10.3 信息架构优先级

```
高优先级（首屏可见）
├── 当前阅读进度
├── AI 辅助入口
└── 词汇学习提醒

中优先级（一次点击可达）
├── 书籍推荐
├── 学习统计
└── 收藏内容

低优先级（设置/更多）
├── 账户管理
├── 偏好设置
└── 帮助中心
```

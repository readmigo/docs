## 4. 特色功能规划

### 4.1 键盘快捷键系统

| 快捷键 | 全局功能 |
|--------|----------|
| `Cmd/Ctrl + K` | 全局搜索 |
| `Cmd/Ctrl + B` | 打开书库 |
| `Cmd/Ctrl + L` | 打开学习中心 |
| `Cmd/Ctrl + ,` | 打开设置 |

| 快捷键 | 阅读器功能 |
|--------|------------|
| `Space` / `PageDown` | 下一页 |
| `Shift + Space` / `PageUp` | 上一页 |
| `←` | 上一章 |
| `→` | 下一章 |
| `F` | 进入专注模式 |
| `Esc` | 退出专注模式 |
| `Cmd/Ctrl + F` | 搜索内容 |
| `Cmd/Ctrl + D` | 添加书签 |
| `Cmd/Ctrl + N` | 添加笔记 |
| `T` | 切换目录 |
| `A` | 切换 AI 面板 |
| `+` / `-` | 调整字体大小 |

| 快捷键 | 复习功能 |
|--------|----------|
| `1` | 忘记了 |
| `2` | 困难 |
| `3` | 一般 |
| `4` | 简单 |
| `Space` | 显示答案 |
| `R` | 播放发音 |

|------|------|
| 单击词汇 | 显示快速释义弹窗 |

| 滚轮 | 滚动页面 |
| Ctrl + 滚轮 | 缩放字体 |

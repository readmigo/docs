# React Native 设计系统

## 设计系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      Design System                               │
├─────────────────────────────────────────────────────────────────┤
│  theme/                                                          │
│  ├── colors.ts          # 颜色定义                               │

│  └── index.ts           # 主题导出                               │

│  components/ui/                                                  │
│  ├── Button.tsx         # 按钮组件                               │

```

## 无障碍设计

```
┌─────────────────────────────────────────────────────────────────┐
│                     Accessibility                                │
├─────────────────────────────────────────────────────────────────┤
│  颜色对比度                                                      │

│  └── 大文字: 最低 3:1                                           │

│  触摸目标                                                        │
│  ├── 最小尺寸: 44 x 44                                          │
│  └── 间距: 至少 8pt                                             │

│  React Native Props                                              │
│  ├── accessible={true}                                          │

│  ├── accessibilityState={{ selected, disabled }}                │
│  └── accessibilityHint="操作提示"                                │
└─────────────────────────────────────────────────────────────────┘
```

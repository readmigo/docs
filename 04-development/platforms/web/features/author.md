## 3. 数据结构

### 3.1 作者信息

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 作者 ID |
| name | string | 作者名 |
| nameOriginal | string | 原名 |
| avatar | string | 头像 URL |
| bio | string | 简介 |
| birthYear | number | 出生年份 |
| deathYear | number | 去世年份 |
| nationality | string | 国籍 |
| worksCount | number | 作品数量 |

### 3.2 作品列表

```
┌─────────────────────────────────────────────────────────────────┐
│                    作品数据结构                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Book                                                           │
│  ├── id: string                                                 │
│  ├── title: string                                              │
│  ├── cover: string                                              │
│  ├── publishYear: number                                        │
│  ├── language: string                                           │
│  ├── genres: string[]                                           │
│  ├── rating: number                                             │
│  └── readersCount: number                                       │
│                                                                  │
│  排序选项                                                        │
│  ├── 出版时间 (升序/降序)                                       │
│  ├── 热门程度                                                    │
│  └── 评分高低                                                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 4. 功能特性

### 4.1 作者简介

| 特性 | 说明 |
|------|------|
| 折叠展开 | 长简介默认折叠 |
| 多语言 | 支持中英文切换 |
| 外部链接 | 维基百科链接 |

### 4.2 作品浏览

| 功能 | 说明 |
|------|------|
| 网格/列表 | 视图切换 |
| 排序 | 多种排序方式 |
| 筛选 | 按类型筛选 |
| 分页 | 无限滚动加载 |

## 6. SEO 优化

### 6.1 元数据

| 字段 | 内容 |
|------|------|
| title | {作者名} 的作品 - ReadMigo |
| description | {作者简介前150字} |
| keywords | 作者名, 作品名列表 |

### 6.2 结构化数据

```
┌─────────────────────────────────────────────────────────────────┐
│                    JSON-LD 结构                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  {                                                              │
│    "@context": "https://schema.org",                           │
│    "@type": "Person",                                           │
│    "name": "作者名",                                             │
│    "description": "作者简介",                                    │
│    "birthDate": "YYYY",                                         │
│    "deathDate": "YYYY",                                         │
│    "nationality": {                                             │
│      "@type": "Country",                                        │
│      "name": "国籍"                                              │
│    }                                                            │
│  }                                                              │
│                                                                  │
```

## 7. 性能优化

### 7.1 数据加载

| 策略 | 说明 |
|------|------|
| 服务端渲染 | 作者基础信息 |
| 客户端加载 | 作品列表分页 |
| 预加载 | 相关作者头像 |

### 7.2 缓存策略

```
┌─────────────────────────────────────────────────────────────────┐
│                    缓存策略                                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  作者信息                                                        │
│  ├── 缓存时间: 1 天                                             │
│  └── ISR revalidate: 3600                                      │
│                                                                  │
│  作品列表                                                        │
│  ├── TanStack Query                                             │

│                                                                  │
│  相关作者                                                        │
│  └── 缓存时间: 1 小时                                           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

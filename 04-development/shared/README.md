## 技术栈概览

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | NestJS (TypeScript) |
| 数据库 | PostgreSQL |
| ORM | Prisma |
| 缓存 | Redis |
| 文件存储 | AWS S3 / Cloudflare R2 |
| 部署 | Fly.io |
| 监控 | Sentry |

# Content Sources

## Overview

Readmigo uses **Standard Ebooks** as the exclusive content source for English public domain books.

---

## Standard Ebooks

**Website**: https://standardebooks.org

**Scale**: 1,000+ high-quality EPUB books

**Documentation**: [standard-ebooks/](standard-ebooks/)

### Key Features

| Feature | Description |
|---------|-------------|
| Quality | Professional typesetting and formatting |
| Covers | Unified, beautiful cover designs |
| Format | Strict EPUB3 standard compliance |
| License | CC0 (completely free to use commercially) |

### Why Standard Ebooks Only

| Reason | Benefit |
|--------|---------|
| Quality First | Best reading experience for users |
| Consistent Format | Simplified parsing and rendering |
| CC0 License | No legal concerns for commercial use |
| Curated Collection | Focus on classic, timeless works |

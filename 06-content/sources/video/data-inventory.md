# 视频内容数据盘点

> 从产品和运营角度，梳理可用于 Readmigo 的视频资源及其价值

---

### 1.1 核心评估维度

| 维度 | 说明 |
|------|------|
| **商业使用** | 是否允许在商业产品中使用 |
| **内容质量** | 与英语学习的相关性和质量 |
| **数据获取** | API 可用性、批量下载支持 |
| **法律风险** | 版权状态、许可证要求 |

---

### 2.1 VOA Learning English

| 项目 | 描述 |
|------|------|
| **网站** | https://learningenglish.voanews.com |
| **定位** | 美国之音英语学习频道 |
| **内容类型** | 新闻、教学视频、纪录片 |
| **商业使用** | ✅ 允许（需注明来源） |
| **法律风险** | ✅ 低（公共领域） |

#### 内容特点

| 特点 | 说明 |
|------|------|
| **语速分级** | Special English (慢速)、Standard English |
| **主题丰富** | 新闻、科技、健康、文化、历史 |
| **配套文本** | 提供对应文本，适合听读同步 |
| **更新频率** | 每日更新 |

#### 对 Readmigo 的价值

| 功能 | VOA 数据 | 价值 |
|------|----------|------|
| **分级听力** | 慢速/标准英语 | 适合不同水平用户 |
| **新闻阅读** | 时事内容 | 提升阅读兴趣 |
| **听读同步** | 音视频+文本 | 增强学习效果 |

### 4.1 版权规则

| 国家 | 公共领域规则 |
|------|--------------|
| **美国** | 1929年前出版 或 版权未续期 |
| **中国** | 作者去世后 50 年 |
| **欧盟** | 作者去世后 70 年 |

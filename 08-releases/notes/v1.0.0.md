# Readmigo v1.0.0 Release Notes

> **发布日期**: 2025 Q1
> **平台**: iOS (Android/Web 后续发布)
> **状态**: 已发布

---

## 版本亮点 | Highlights

首个正式版本，专注于英文经典阅读体验。

The first official release, focused on English classics reading experience.

---

## 功能清单 | Feature List

### 书架与发现 | Library & Explore

| 功能 | 说明 |
|------|------|
| 书架展示 | 用户已添加书籍的网格展示 |
| 发现页 | 浏览和发现新书 |
| 书籍详情 | 书籍信息、AI 功能入口 |
| 榜单浏览 | 各类推荐榜单 |
| 分类浏览 | 按分类浏览书籍 |
| 阅读进度 | 继续阅读、进度同步 |
| 书籍搜索 | 搜索书名、作者 |
| 离线下载 | 下载书籍供离线阅读 |

### 阅读器 | Reader

| 功能 | 说明 |
|------|------|
| EPUB 渲染 | 高质量电子书渲染 |
| 翻页手势 | 点击/滑动翻页 |
| 文字选择 | 长按选中文字 |
| AI 查词 | 上下文词汇解释 (计划废弃) |
| 阅读设置 | 字体、主题、行距 |
| 进度同步 | CFI 定位实时同步 |
| TTS 朗读 | 文字转语音 |

### 城邦 | Agora

| 功能 | 说明 |
|------|------|
| 信息流 | 金句信息流展示 |
| 帖子卡片 | 作者头像、金句内容、来源信息 |
| 点赞 | 点击/双击点赞 |
| 分享 | 生成金句图片分享 |
| 内容管理 | 不感兴趣、屏蔽作者、举报 |

### 订阅 | Subscription

| 功能 | 说明 |
|------|------|
| 付费墙 | 订阅计划展示 |
| 应用内购买 | 订阅购买流程 |
| 订阅管理 | 查看/取消/恢复订阅 |
| 权益管理 | Premium 功能控制 |

### 其他功能 | Others

| 功能 | 说明 |
|------|------|
| Apple/Google 登录 | 第三方账号登录 |
| 游客模式 | 无需注册即可体验 |
| 生词本 | 收藏生词、复习 |
| 数据同步 | 多设备数据同步 |

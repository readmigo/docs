- 离线下载：下载书籍供离线阅读

**阅读器**
- EPUB 渲染：高质量电子书渲染
- 翻页手势：点击/滑动翻页
- 文字选择：长按选中文字
- AI 交互：查词、解释、翻译
- 阅读设置：字体、主题、行距

- 信息流：金句信息流展示
- 帖子卡片：作者头像、金句内容、来源信息
- 点赞：点击/双击点赞

- 300 本电子书

### English Version

```markdown
## Readmigo v1.0.0 - Initial Release

The first official release focused on English classics reading experience.

### New Features

**Library & Explore**
- Library: Grid display of user's added books
- Explore: Browse and discover new books
- Book Details: Book info and AI feature entry
- Rankings: Various recommendation lists
- Categories: Browse books by category
- Reading Progress: Continue reading with sync
- Book Search: Search by title or author
- Offline Download: Download books for offline reading

**Reader**
- EPUB Rendering: High-quality ebook rendering
- Page Turn Gestures: Tap/swipe to turn pages
- Text Selection: Long press to select text
- AI Interaction: Look up, explain, translate
- Reading Settings: Font, theme, line spacing
- Progress Sync: Real-time CFI-based sync
- TTS: Text-to-speech reading

**Agora (Social Feed)**
- Feed: Quote-based social feed
- Post Cards: Author avatar, quote content, source info
- Like: Tap/double-tap to like
- Share: Generate quote images to share
- Content Management: Hide, block author, report

**Subscription**
- Paywall: Subscription plan display
- In-App Purchase: Subscription purchase flow
- Subscription Management: View/cancel/restore
- Entitlement Management: Premium feature control

**Others**
- Apple/Google Sign-in
- Guest Mode
- Vocabulary Book
- Data Sync

### Content Scale

- 300 eBooks
- 100 Authors

### Known Limitations

- Audiobooks not supported yet
- Agora comments not supported yet
```
